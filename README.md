# NFC Cookie
### https://dionhobdy.github.io/nfccookie

![NFC Cookie Web View](https://raw.githubusercontent.com/dionhobdy/nfccookie/refs/heads/main/Assets/1.png)

### DESCRIPTION
NFC Cookie is a simple fortune cookie simulator that generates random fortunes and sets of lucky numbers. The reason why this was made is to scan NFC stickers hidden inside of origami fortune cookies and load this webside.
